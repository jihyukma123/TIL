# 뭘 해야됨?

**Master the foundations, become an expert**

- data structure
- algorithm
- programming concepts

**Think about systems, like you are a system architect**
-> meet the expectations of a senior engineer as soon as possible(aim higher)
-> think more about designing systems that are more reliable and scalable

**Go FullStack**

- 개발의 다양한 분야뿐만 아니라,
- 프로덕트 생성의 과정에서 협업하는 다양한 분야에 대해서 이해가 필요함
  - 디자인, 프로덕트, 데이터, PM 등
- be prepared to wear multiple hats

**Practice communication and collaborations**

Learn to work with people

If you can explain and connect, it will set you apart.
-> 이해한 것을 말로 풀어내고, 맥락을 엮어낼 수 있는 사람은 자연히 두각을 나타낸다.

**Adaptiveness**

Tools change, principles don't

learn how to learn.
