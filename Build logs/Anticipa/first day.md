# first day trying to build an app

디데이용 앱을 만들어보자는 생각을 가지고 시작했음.

근데 최근에 OpenAI의 Harness Engineering이라는 글을 읽어서, 나도 이런 느낌으로 완전히 AI Agent한테 코딩을 맡길 수 있는 구조를 설계하고 시작해보려고 고민을 막 시작했음.

근데 한 3시간 정도 고민해보는 과정을 통해서 느낀 점은...

1. 한 번도 완전히 Full Agentic Coding Architecture 를 설계해보지 않은 사람이 다 깔고 시작하기는 쉽지 않다. 그냥 어디서부터 어떻게 손대야할지 감이 안옴. AGENTS.md를 시작점으로 삼아야 한다 정도만 알겠는 느낌..?(그래도 이건 알고 있으니까 다행이다.ㅎㅎ)

2. 내가 문제를 분해해서 내가 필요한 아키텍처를 짤 줄 알아야 한다.

3. 2번을 위해서는, Harness Engineering 과 같은 글에서 그들이 실제로 어떻게 했는지 그 구체적인 implementation이 아니라, idea를 이해해야 한다. `progressive disclosure`가 뭔지, 왜 그렇게 하는건지, 어떻게 하는건지 등등...

4. 마찬가지로, idea를 이해하려면 내가 사용하는 기술이 가지는 특성에 대한 이해가 필요하다. Codex, Claude Code 같은 친구들이 어떤 특성을 가지고 동작하는지에 대한 이해가 필요함. 결국 LLM기반으로 돌아가기 때문에 이해해야되는 것들, 구체적으로 지시하고 조금만 시키고, 하지말아야 하는 것도 명시하고 등등...왜 그래야하는지에 대한 이해가 필요하다.

5. 문제를 분석하는 것에서 시작하는건 예전하고 달라진게 없다. 뭘 하고 싶은건지가 뾰족할수록 잘 지시할 수 있고 잘 해줄 가능성이 높아진다.
