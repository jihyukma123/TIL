# 3월 3일

문득 그런 생각을 했다.

'너무 잘할려고 할 필요 없다.'

무슨 말이냐면, 너무 잘해야된다는 생각에 내가 했던 일을 '잘 모르겠다'라고 하던지 아니면 '별거 아니다'라고 할 필요가 없다는 생각이 들었다는 말이다.

Agent Builder 관련 기술 검증을 수행한 후에 문서를 작성해서 공유했는데, 나는 약간 테스트를 충분히 했는지에 대해서 확신이 없는 상태에서 일단 배우고 느낀 내용을 작성해서 공유했는데 도움이 많이 되었다는 피드백을 받았다.

당연히 모든 걸 다 파악해서 공유할 수는 없는 노릇인데.... 할 수 있는만큼 하고 피드백을 받아서 필요한 부분을 보완하면 된다고 생각하면서도 그게 잘 체득이 안되는 것 같다.

잘 했어. 또 조사해보면서 알아낸 내용을 공유해서 모두의 지식 범위를 확장하는데 조금이라도 도움이 되면 그걸로 되었다.
